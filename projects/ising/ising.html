2D Ising Model

This simulation models an idealized 2D magnetic material using the Ising model. You can adjust the lattice size and system temperature to observe the phase transition between a ferromagnetic and paramagnetic state.

To control the simulation, you can:
→ Adjust the lattice size
→ Change the system temperature
→ Use the start button to run the simulation
→ Click the reset button to set a new random initial state
