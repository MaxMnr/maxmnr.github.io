This interactive tool allows you to explore the Mandelbrot set. You can zoom in by dragging a rectangle on the canvas with your mouse and adjust the number of iterations using the slider to refine the set's details.

To navigate the zoom levels, you can:
→ Use the back button to return to the previous zoom level
→ Click the reset button to return to the initial zoom level
